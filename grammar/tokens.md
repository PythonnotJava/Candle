## 分隔符

| 符号 | 用途 |
|------|------|
| `(` `)` | 分组、参数列表、条件 |
| `{` `}` | 块、类体、export 列表 |
| `[` `]` | 列表字面量、下标 |
| `<` `>` | 泛型参数 |
| `;` | 语句结束 |
| `,` | 列表分隔 |
| `.` | 成员访问、包路径 |
| `:` | map 键值对 |
| `?` | 可空类型标记 |
| `@` | 运算符重载 |
| `=>` | 箭头（匿名函数） |

## 字面量

### 整数
- 十进制: `42`, `0`, `1000000`
- 十六进制: `0xFF`, `0x1A3F`
- 二进制: `0b1010`, `0b11110000`
- 八进制: `0o777`, `0o12`

### 浮点数
- 基本: `3.14`, `0.5`
- 科学计数法: `1.5e10`, `2.0E-3`

### 字符串
- 普通: `'hello'`, `"world"`
- 格式化: `f'name is {name}'`
- 原始: `r'\n is literal'`

### 布尔
- `true`, `false`

### 空值
- `null`

## 注释

- 单行: `// comment`
- 多行: `/* comment */`

## 特殊语法标记

| 标记 | 位置 | 含义 |
|------|------|------|
| `?` | 类型后缀 | 可空类型 `int?` |
| `!` | 表达式后缀 | 非空断言 `value!` |
| `!` | 表达式前缀 | 逻辑取反 `!flag` |
| `\|` | 类型之间 | 联合类型 `int \| double` |
| `@` | 方法前缀 | 运算符重载 |
| `f` | 字符串前缀 | 格式化字符串 |
| `r` | 字符串前缀 | 原始字符串（不转义） |
| `=>` | 表达式中 | 匿名函数箭头 |
